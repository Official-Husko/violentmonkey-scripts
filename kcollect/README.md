# Kcollect Scripts

Here is a script for [Kcollect][2].

## Remove Watermark
Removes the watermark from Kcollect cards.

### Installation
1. Install [Violentmonkey][5] (Others like [Tampermonkey][6] work too but are ***not recommended***)
2. Install the [Script][7]
