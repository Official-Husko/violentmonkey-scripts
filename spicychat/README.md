# Spicychat Scripts

Here is a script for [Spicychat][2].

## No Blurry Avatar
This simply makes blurry avatars clearer and high-res.

### Installation
1. Install [Violentmonkey][5] (Others like [Tampermonkey][6] work too but are ***not recommended***)
2. Install the [Script][7]
